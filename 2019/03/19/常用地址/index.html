解决方案汇总

go basic

  • 基础数据类型
    • 有符号整数类型：int8(-128 到 127), int16(-32768 到 32767), int32(-2147483648 到 2147483647), int64（±9.2e18）
    • 无符号整数类型：uint8, uint16, uint32, uint64. 一般情况下不会使用无符号数，在减法运算时可能出问题
    • 浮点数：float32(math.MaxFloat32=3.4e38, math.MinFloat32=1.4e-45), float64(math.MaxFloat64=1.8e308, math.MinFloat64=4.9e-324)
    • boolean: true/false
    • 字符串：不可变字符串序列，java中字符串也是不可变量
    • 常量：const
  • 集合类型
    • 数组：长度不可变

      var a [3]int
      var a [3]int = [3]int{1,2,3}
      var a := […]int{1,2,3,4}
    • slice: 相同类型元素的可变长度序列，底层用数组来表示 ，长度变化时可能涉及底层元素的迁移复制（就是典型的java数组扩容复制）

      var s := make([]int, len, cap)
      var s := make([]int, len)
      s = append(s, 1)
    • map：散列表的引用

      ages := make(map[string]int) // name-age map
      ages := map[string]int{
        “alice”: 20,
        “collins”: 19,
      }
    • 结构体struct
  • 指针和地址。go全部使用传值参数，被调用的函数收到的是主调函数复制的变量的副本，修改这个副本不会对主调函数中的值产生任何影响，因此如果需要修改参数的值或因为参数过大想要避免复制，就可以在参数定义中使用指针参数
    • *ptr定义一个指针变量
    • &var获取一个变量的地址，可以复制给一个指针变量

      val := 1
      ptr := &val
  • 并发：go routine和channel
    • 与线程的区别包括：①栈长度可变（2K-1G）②M:N调度
    • Concurrency is not parallism
  • 包管理：import语句和govenor
    • 不需要再git clone，直接用go get来完成源码的拉取和安装执行
    • 一种内部惯例：仔细维护vendor/目录，将所有的vendor相关变更也提交到代码库
    • 这种模式下：

      // 不使用选项，下载后会自动完成编译和打包
      go get xxx.gitlab.xxx/yyy/zzz
      // -d选项只下载源码，不执行安装
      go get -d xxx.gitlab.xxx/yyy/zzz
  • 常用包及功能
    • fmt包 打印输出控制
    • strings/bytes/strconv/unicode包 用于字符串操作
    • math包 封装基础数学函数
    • sync包 用于并发锁控制
    • json包 用于json序列化/反序列化等操作
    • io/bufio包 用于管理输入输出流Reader/Writer
    • sql/driver包 数据库访问
